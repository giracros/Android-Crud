Android-Crud
============

Este proyecto fue realizado con los siguientes elementos:
---------------------------------------------------------
1. AndroidStudio (Beta)
2. SQLite3
3. Gradle

El proyecto es un CRUD para la administracion de una biblioteca
---------------------------------------------------------------

1. Crear 
2. Consultar
3. Modificar
4. Eliminar

### Instalacion ###

+ Contar con AndroidStudio
+ SDK 16 o posterior
